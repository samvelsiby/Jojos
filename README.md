# Jojo's Pastry Shop

A beautiful, responsive website for Jojo's Pastry Shop featuring:

- Initial splash animation with logo reveal
- Responsive design for all device sizes
- Beautiful UI with smooth animations using Framer Motion
- Featured pastries section with consistent image sizing
- Online ordering functionality

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Vite

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Features

- Smooth animations and transitions
- Mobile-responsive design
- Shopping cart functionality
- Modern UI with a clean aesthetic
